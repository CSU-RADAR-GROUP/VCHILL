If your mouse (or other pointing device) does not have a middle button, the recentering operation is also available from the context menu.

When the data readout function is requested in a Plot window, all open windows will replot the ray containing that value (if possible). In this way, it is possible to use a Plot window to "aim" an AScope view of the data.
